# Esbuild Plugin Metadata

A plugin for [esbuild](https://esbuild.github.io/) that writes an `entrypoints.json` manifest for builds.

[![NPM](https://img.shields.io/npm/v/@chialab/esbuild-plugin-metadata.svg)](https://www.npmjs.com/package/@chialab/esbuild-plugin-metadata)

## Documentation

Read the documentation at [chialab.github.io/rna](https://chialab.github.io/rna/guide/esbuild-plugin-metadata).
