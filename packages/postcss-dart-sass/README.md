# Postcss Dart Sass

A postcss plugin that uses [dart sass](https://sass-lang.com/dart-sass) to transpile scss files.

[![NPM](https://img.shields.io/npm/v/@chialab/postcss-dart-sass.svg)](https://www.npmjs.com/package/@chialab/postcss-dart-sass)

## Documentation

Read the documentation at [chialab.github.io/rna](https://chialab.github.io/rna/guide/postcss-dart-sass).
